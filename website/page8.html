JavaScript Const

کلمه کلیدی const در ES6 (2015) معرفی شد.
متغیرهای تعریف شده با const را نمی توان مجدداً اعلام کرد.
متغیرهایی که با const تعریف شده اند را نمی توان دوباره اختصاص داد.
متغیرهای تعریف شده با const دارای Block Scope هستند.

چه زمانی از JavaScript const استفاده کنیم؟

به عنوان یک قانون کلی، همیشه یک متغیر را با const اعلام کنید، مگر اینکه بدانید مقدار تغییر خواهد کرد.
از const هنگام اعلام استفاده کنید:
1-یک آرایه جدید
2-یک شی جدید
3-یک عملکرد جدید
4-یک RegExp جدید

اشیا و آرایه های ثابت

کلمه کلیدی const کمی گمراه کننده است. مقدار ثابتی را تعریف نمی کند. یک ارجاع ثابت به یک مقدار را تعریف می کند. به همین دلیل شما نمی توانید:

  • یک مقدار ثابت را دوباره تخصیص دهید
  • یک آرایه ثابت را دوباره اختصاص دهید
  • یک شی ثابت را دوباره اختصاص دهید

    • ولی شما می توانید:

  • عناصر آرایه ثابت را تغییر دهید
  • ویژگی های جسم ثابت را تغییر دهید

اپراتورهای واگذاری جاوا اسکریپت

عملگرهای انتساب مقادیری را به متغیرهای جاوا اسکریپت اختصاص می دهند.
اپراتورها مثال ها مثل
= x = y x = y
+= x += y x = x + y
-= x -= y x = x - y
*= x *= y x = x * y
/= x /= y x = x / y
%= x %= y x = x % y
<<= x <<= y x = x << y
>>= x >>= y x = x >> y
>>>= x >>>= y x = x >>> y
&= x &= y x = x & y
^= x ^= y x = x ^ y
|= x |= y x = x | y
**= x **= y x = x ** y


عملگر **= بخشی از ECMAScript 2016 است.

نمونه های تکلیف

عملگر انتساب = مقداری را به یک متغیر اختصاص می دهد.

محاسبات

عملگر انتساب += یک مقدار به یک متغیر اضافه می کند.

عملگر *= یک متغیر را ضرب می کند.

تخصیص /= یک متغیر را تقسیم می کند.

عملگر انتساب %= باقیمانده ای را به یک متغیر اختصاص می دهد.
